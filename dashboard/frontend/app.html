                  Daily Leaderboard

                  The Daily Leaderboard is the same chart + standings UI as Competition, but each entry is an hourly backtest over the last completed US weekday. Baseline and index lines refresh automatically on load. LLM entries are added when the daily deploy job runs.

                  Competition Leaderboard

                  Preseason Standings are a backtest leaderboard, not live trading. Each entry is an hourly backtest over 2026-04-15 → 2026-05-15 (one-month window), comparing provided LLM models, baseline strategies, and market indices. Official rank is by final portfolio value.
