Orchestration Framework
=======================

The **FinAgent Orchestration Framework** (under ``orchestration/`` in the repository) implements multi-agent coordination for agentic trading: DAG planning, specialized agent pools, memory systems, and protocol-based communication (MCP, A2A).

Browse by topic:

- :doc:`introduction` — motivation, architecture overview, prompt design
- :doc:`workflow_and_execution` — training workflow, execution, orchestrator, tutorials
- :doc:`agent_pools` — data, alpha, risk, portfolio, memory, and transaction-cost agents
